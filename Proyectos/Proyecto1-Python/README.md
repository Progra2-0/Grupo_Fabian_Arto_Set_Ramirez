# Proyecto 1- Python – Sistema de Análisis Robótico

## Estructura del Proyecto
- `main.py` — Contiene la lógica principal de la simulación. No debe ser modificado.
- `robot_base.py` — Clase base `RobotBase` con la lógica común, encapsulamiento y cinemática de los robots.
- `modelos_robot.py` — Clases de robots específicos (Triciclo, Oruga, Dron) que heredan de `RobotBase`, demostrando polimorfismo.
- `analisis.py` — Función para analizar los datos de la simulación utilizando NumPy.
- `visualizacion.py` — Función para graficar los resultados utilizando Matplotlib.

## Ejecución
python main.py
